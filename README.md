# sitemapgenerator

Python3+.

## Install
```
pip install sitemapgenerator
```




-----------------------------------------------------------

## Development

### Setup

#### Set up virtualenv
pyvenv env
source env/bin/activate
```

#### Install requirements
```
pip install -r requirements.txt
```

#### Update requirements
```
pip install -r requirements-to-freeze.txt --upgrade
pip freeze > requirements.txt
```
